번호가 없을 때에는 - * + 를 붙이면 글머리 기호가 자동으로 붙음

+ 텍스트 굵게는 **굵게**  (별별로 감싸, 별별 대신 언더바 두개로 해도 됨.)

+ 기울임체는 *기울임* (별하나 앞뒤 감싸)

+ 굵은 기울임체는 ***굵게기울여*** (별별별 로 감싸)

+ 최소선은 ~취소~ (물결로 감싸)

> 인용문은 앞에 꺽쇠 (>)
>> 인용문 안에 또 다른 인용문 삽입시 꺽쇠두개 (>>)

소스코드는 앞에 쩜 그레이브키'

add 함수는 이렇게 사용해 'function add(x,y)

여러줄 소스코드는 쩜쩜쩜 과 쩜쩜쩜 으로 감싸

'''javascript
<script>
   ...
</script>
'''
